다이소 안성센터 WorkTracker

업무 관리 시스템을 앱으로 설치하여 더 편리하게 사용하세요!

📋 설치 방법
🌐 Chrome / Edge 브라우저
  1. 브라우저 주소창 오른쪽의 설치 버튼을 클릭하세요
  2. 또는 Ctrl + Shift + I를 눌러 개발자 도구를 열고
  3. Application 탭 → Manifest → Install 버튼을 클릭하세요
🍎 Safari (iOS)
  1. 하단의 공유 버튼을 탭하세요
  2. 홈 화면에 추가를 선택하세요
  3. 추가를 탭하여 설치를 완료하세요
📱 Samsung Internet
  1. 메뉴 버튼을 탭하세요
  2. 앱 설치를 선택하세요
  3. 설치를 탭하여 완료하세요
✨ 앱 설치 후 제공되는 기능
  • ✅ 브라우저 주소창 없이 앱처럼 사용
  • ✅ 홈 화면에서 바로 접근 가능
  • ✅ 오프라인에서도 기본 기능 사용
  • ✅ 빠른 로딩과 부드러운 사용자 경험
  • ✅ 푸시 알림 지원 (향후 업데이트)
